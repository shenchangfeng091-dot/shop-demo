# 潮物集市 — 购物网站 Demo

一个纯前端实现的电商购物网站演示项目。

## 功能

- 18 件商品，6 大分类（数码/服饰/家居/美妆/运动/食品）
- 分类筛选 & 实时搜索
- 价格/销量排序
- 商品详情弹窗
- 购物车侧边栏（增删改数量、合计计算）
- 结算演示
- 响应式布局（手机/电脑自适应）

## 技术栈

纯 HTML + CSS + JavaScript，无框架、无后端依赖。

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 部署到 GitHub Pages

1. 将本仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main` / `root`
5. 保存后等待 1-2 分钟，即可通过 `https://你的用户名.github.io/仓库名/` 访问
